== Description ==

[youtube https://www.youtube.com/watch?v=iGZZrxwV4RQ]

**DEMO**: <https://demo.add-ons.org/demo-international-phone-field/>

The Telephone field for the Elementor Forms plugin helps you in creating a country drop-down list with country flags. The tag field will automatically add countries name in standard drop-down field of Elementor Forms.

It supports all modern browsers and touch based devices.


== Features ==
- **Easy to use**
- **Phone number validation**: Ensure submitted phone numbers are real numbers. No more throwaway numbers. Less typos.
- **Use Phone meta in conditional logic.**: Trigger conditional logic based on captured metadata.
- **Capture phone meta**: Capture phone type, location, carrier, and more.
- **Filtered and preferred countries.**: Filter which countries are available for selection and place preferred countries at top of selectable list.
- **Auto-detect default country.**: Automatically detect the country code by visitor location or set a default country.
- **Selectable country codes with flags.** Customers can easily select their country code and will appreciate country flags as a visual aid.

== Installation ==
**Normal installation**

1. Download the telephone-field-for-Elementor-Forms.zip file to your computer.
2. Unzip the file.
3. Upload the `telephone-field-for-Elementor-Forms` directory to your `/wp-content/plugins/` directory.
4. Activate the plugin through the 'Plugins' menu in WordPress.
5. Document: https://add-ons.org/document-elementor-forms-international-phone-masking-field/


== Frequently Asked Questions ==

= Why is the country flag image blurred? =
The blurred national flag image is due to other plugins optimizing the image incorrectly. Please contact us to fix it.

= Why does the pop-up screen disappear after completing country selection on mobile? =
This is a popup feature of elementor, to fix the error you enable Prevent Closing on Overlay in the popup settings.
<https://add-ons.org/document-elementor-forms-international-phone-masking-field/#document-h2-8>
